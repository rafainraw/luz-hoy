Política de privacidad

Última actualización: 4 de julio de 2026

Luz Hoy no crea cuentas propias, no contiene publicidad y no opera un servidor propio que almacene perfiles de usuario.

1. Responsable y contacto

Luz Hoy es una aplicación para consultar precios eléctricos, calcular costes y recibir avisos. Para consultas sobre privacidad o soporte: rafa@rafainraw.com.

2. Precios y conexiones

La aplicación consulta precios eléctricos de eSIOS y, cuando es necesario, utiliza Elecz como fuente alternativa. Estas consultas se realizan mediante conexiones cifradas y no incluyen una cuenta propia de Luz Hoy.

3. Cámara y reconocimiento de texto

Las fotografías de etiquetas se procesan en el dispositivo mediante ML Kit. La imagen temporal se guarda en la caché privada de la aplicación y se elimina al terminar el procesamiento. Las fotos y el texto reconocido no se envían a un servidor de Luz Hoy.

4. Búsqueda opcional de modelos

Si el usuario pulsa una opción de búsqueda online, la marca, el modelo o la consulta escrita se envía a DuckDuckGo para localizar estimaciones públicas de consumo. Esta función es opcional y solo se ejecuta por acción del usuario.

5. Datos almacenados localmente

Las preferencias, consumos guardados y programación de avisos permanecen en el dispositivo. Las copias de seguridad de los datos de la aplicación están desactivadas.

6. SDK y diagnósticos

ML Kit puede tratar información técnica, identificadores por instalación y diagnósticos conforme a las políticas de Google, con el fin de prestar, proteger y mejorar sus servicios.

7. Permisos

  • Cámara: reconocer texto de etiquetas.
  • Notificaciones y alarmas: mostrar avisos locales solicitados por el usuario.
  • Internet: consultar precios y realizar búsquedas opcionales.

8. Eliminación

Los datos locales pueden eliminarse desde los ajustes de Android o desinstalando la aplicación.

9. Cambios

Esta política puede actualizarse para reflejar cambios en la aplicación o en los servicios utilizados. La fecha de la versión vigente aparece al comienzo del documento.
